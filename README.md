# claude-plugins

Claude Code plugin marketplace by neox-d.

## Add this marketplace

```
/plugin marketplace add neox-d/claude-plugins
```

## Install plugins

```
/plugin install erpnext-timesheet@neox-d-plugins
```

## Plugins

| Plugin | Description |
|--------|-------------|
| [erpnext-timesheet](https://github.com/neox-d/claude-erpnext-timesheet) | Auto-fill ERPNext timesheets from Claude conversation history |
